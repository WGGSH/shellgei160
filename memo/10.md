# 10

```.sh
cat headings.md | awk '{if($1=="#"){printf "%s\n===\n", $1}else if($1=="##"){printf "%s\n---\n", $2}else{print $0}}'
```

## input

```.md
# AAA

これはAAAです

# BBB

これはBBBです。
楽しいですね。

## CCC

これはCCCCです

## DDD

これはDDDです
```


## output
```.md
#
===

これはAAAです

#
===

これはBBBです。
楽しいですね。

CCC
---

これはCCCCです

DDD
---

これはDDDです
```
