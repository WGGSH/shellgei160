# 問7

```.sh
cat kakeibo.txt | awk 'BEGIN{sum=0}{if($1>20191001 && $2 !~ "^\\*"){tax=0.1}else{tax=0.08}{sum+=int($3*(1+tax))}}END{print sum}'
```

## format

```
20190901 ゼロカップ大関 10000
20190902 * キャベツ次郎 130
```

のようになってる

## メモ
- 総和を記録する `sum` を用意
- $1, $2 を元に消費税率 `tax` を計算
  - 2019/10/01 より前の場合,すべて8%
  - 2019/10/01 以降の場合,
    - `*` が先頭についていれば 8%
    - `*` が先頭についていなければ 10%
- `sum` に += で足してく
- `END{}` で集計結果を出力
